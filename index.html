Mean Shift Algorithm for Clustering


  • Mean Shift is a algorithm for clustering or tracking task

  • To Start with, Mean shift calculate the mean value of data in a circular windows

  • The centeroid of data decides the direction for windows to shift

  • Click on the circular window to create data point, see how centeroid affect by data distribution

    • (You can also drag the data point)

    What Can you find?

    When the iteration times is not enough, the algorithm may not converge

    So we usually stop the iteration based on the renew of the centeroid

    When the bind_width is too small, it has too many meangingless clusters

    When the bind_width is large, it converges to some unfitted result

    Hover on the Image to see the segmentation result

  • The Mean Shift algorithm run on the RGB data of graphs

  • The algorithm groups the similar color together in RGB space

  • This is useful for image segmentation based on colour or texture levels
